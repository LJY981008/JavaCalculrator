# JAVA로 구현하는 계산기
## 프로젝트 소개
JAVA 문법연습을 하기위해 시작한 계산기 만들기

## 프로젝트 구조(패키지 기준)
- lv1 : 클래스 없이 기본적인 연산을 수행할 수 있는 계산기 만들기
- lv2 : 클래스를 적용해 기본적인 연산을 수행할 수 있는 계산기 만들기
- lv3 : Enum, 제네릭, 람다 & 스트림을 이해한 계산기 만들기
